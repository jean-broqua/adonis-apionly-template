# Adonis API Only Template

Pre configured API template usigin [AdonisJS](https://adonisjs.com/).

## How to setup:

`yarn` or `npm install`

### Configuration Options:

- Eslint
- Prettier
- WebToken Auth
- PostreSQL Database

### Included

- User Model / Migration
- Register / Login Routes
- User Factory
- Regiter / Login Tests
- Redis
- docker-compose with PostreSQL and Redis.
